1-2 CSV에서 theta_hat 평균값을 구하고 모델의 균형각과 비교한다.

% MATLAB Command Window data = readtable('off_mode_upright.csv'); theta_hat_measured = mean(data.theta_hat) * 180/pi load('dynamic_properties.mat'); load('dynamics_functions.mat'); model = Pol(properties, dynamic_functions); theta_eq_ref = model.get_theta_eq(0.2, 0) * 180/pi % → -0.55° 근방이어야 함 delta = theta_hat_measured - theta_eq_ref % → 이 값이 CoM 편차
delta 범위판단조치
|delta| < 1° CoM 정상 OK 게인 그대로 사용
1° ≤ |delta| < 2° 경계선 HOLD Run 모드 테스트 후 판단
|delta| ≥ 2° CoM 쏠림 수정 필요 아래 CoM 수정 파이프라인 실행
▸ CoM 수정 파이프라인 (|delta| ≥ 2° 일 때)
